# Glossary

- **ADC**: Autonomous Development Constitution. The project context, rules, standards, planning, and knowledge structure under `.adc/`.
- **CGA**: Context Graph Agent. Local-first graph context service used for indexed retrieval and project activity aggregation.
- **CGA-Relay**: Local project relay used to aggregate official project change information into CGA when credentials are configured.
- **QS**: QuickSearch.
- **TSG**: Troubleshooting guide or team-specific support guide, based on current folder naming.
- **SOP**: Standard operating procedure.
- **CASE**: Case-related document category in the current search type list.
- **TEAM**: Team-folder search category, currently routed to `Orcas_Main\team`.
- **Mapped Drive**: A Windows drive letter mapped to a shared network location.
- **Generated Index**: Reproducible JSON output created by indexing scripts; not canonical source unless explicitly approved.
