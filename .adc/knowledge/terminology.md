# Terminology

Use these names consistently:

- Refer to the project as `QuickSearch` on first mention and `QS` after that.
- Refer to `src/QuickSearch.ps1` as the main QS UI entry script.
- Refer to `src/QuickSearch.Support.ps1` as the QS support script for shared config, search, preview, Index settings, and indexing functions.
- Refer to `src/data/index.json` as generated file index data when the re-index action is used.
- Refer to `src/settings/config.json` and `src/profiles/*.profile.json` as QS configuration files; `default.profile.json` is the default profile.
- Refer to `Orcas_Main\TSG-SOP` and `Orcas_Main\team` as configured mapped-drive targets, not as repository-owned folders.
